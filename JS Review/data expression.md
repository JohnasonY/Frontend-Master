# Identifier

*JavaScript* identifier (variable name, function name, parameter name) need to follow these rules:

- allows number, letter, underline, $ symbol
- can not start with number
- can not conflict with keywords
- recommend using camelCase naming rule

***

# Escape Character

| Escape Characters | Definition |
| ----------------- | ---------- |
| `\'`              | symbol '   |
| `\"`              | symbol "   |
| `\r`              | Enter      |
| `\n`              | next line  |

***

# Data Type

**Primitive data type**: number, string, boolean, null, undefined

**Reference data type**: object (regular object, array, function)

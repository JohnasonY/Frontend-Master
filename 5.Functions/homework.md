# General Function Writing

Create a new JavaScript file and implement the following functions:

1. Write a function that checks whether a given number is odd.
Suggested function name: isOdd

2. Write a function that checks whether a given number is prime.
Suggested function name: isPrime

3. Write a function that calculates the sum of all elements in an array.
Suggested function name: sumOfArray

4. Write a function that returns the maximum value in an array.
Suggested function name: maxOfArray

5. Write a function that returns the minimum value in an array.
Suggested function name: minOfArray

6. Write a function that checks whether an array is sparse (i.e., contains empty slots).
Suggested function name: hasEmptyInArray

7. Write a function that determines whether a given year is a leap year.
Suggested function name: isLeap

8. Write a function that returns the number of days in a specific month of a given year.
Suggested function name: getDays

9. Write a function that finds the number(s) that occur most frequently in a numeric array, along with their frequency.
Suggested function name: getTopFreqInArray
